# Погода (weather)

Плагин Usee: текущая погода по городу через [Open-Meteo](https://open-meteo.com)
— бесплатный API **без ключа**.

## Карточки
- **gauge «температура»** — текущая t° как `number` со шкалой-заливкой
  (метрика −30…45 °C, имперская −20…115 °F).
- **status «условия»** — иконка погоды + описание (line1), «ощущается / влажность
  / ветер» (line2), t° в бейдже.

Иконки (☀️ ясно · ⛅ переменная · ☁️ пасмурно · 🌫 туман · 🌧 дождь · 🌨 снег ·
⛈ гроза) рисуются **процедурно в RGB565 прямо в плагине** — без сети и внешних
ассетов, фон под цвет плитки устройства. 7 иконок (лимит — 8 на плагин).

Что показывать — настройка **«Карточки»** (обе / только gauge / только status).

## Настройки
| Ключ | Тип | Назначение |
|------|-----|-----------|
| `city` | string | Город (свободный ввод, напр. «Москва») — резолвится геокодингом Open-Meteo в рантайме; результат/ошибка видны в статусе плагина |
| `units` | select | `metric` (°C · м/с) или `imperial` (°F · mph) |
| `card` | select | `both` / `gauge` / `status` |
| `label` | string | Подпись карточек (по умолчанию — имя города) |
| `intervalMin` | number | Период опроса, 1–240 мин (по умолчанию 15) |
| `width`, `minW`, `maxW` | — | Ширина карточек (как в остальных плагинах) |

## Права
`net: ["*.open-meteo.com"]` — только геокодинг и прогноз Open-Meteo. Ключей и
секретов нет.

## Данные Open-Meteo
- Геокодинг: `https://geocoding-api.open-meteo.com/v1/search?name=<город>`
- Прогноз: `https://api.open-meteo.com/v1/forecast?...&current=temperature_2m,`
  `apparent_temperature,relative_humidity_2m,weather_code,wind_speed_10m`

`weather_code` (WMO) переводится в русское описание внутри плагина.

## Локальная проверка
Скопировать `manifest.json` и `plugin.js` в
`%AppData%\PcMonitorAgent\plugins\weather\1.0.0\`, включить плагин в агенте и
задать город. Публикация в реестр (`plugins/index.json`) — отдельный шаг
(залить бандл в usee-plugins и прописать `sha256`/URL, как у прочих плагинов).
