# Home Assistant — плагин Usee

Полный паритет встроенной интеграции HASource: карточки по биндингам сущностей
(gauge / meters / status, авто-выбор), условия видимости, «—» при недоступности
с 20-секундным удержанием последнего значения, тосты из событий `usee_notify`,
staleness, heartbeat и реконнект с бэкоффом.

## Установка

Из каталога Usee («Настройки → Плагины → Каталог») или вручную: положите
`manifest.json` + `plugin.js` в
`%AppData%\PcMonitorAgent\plugins\home-assistant\1.0.0\`.

## Настройки

- **Адрес сервера** — `http://homeassistant.local:8123` (или ws-URL).
- **Вход через браузер** — OAuth-вход; агент хранит refresh-токен зашифрованным
  и плагин сам обменивает его на короткий access-токен.
- **Long-lived токен** — альтернатива входу (профиль HA → Security → токены).
- **Сущности** — список биндингов; каждый: сущности (через запятую — 2+ дают
  meters), тип карточки, подпись/единица, min/max для шкалы, бейджи для status,
  ряд/порядок/ширина и условие видимости (оператор + значение).

Тосты: событие `usee_notify` с `title`, `message?`, `kind?`
(notice|done|critical|achievement), `ttlMs?` — например из автоматизации:

```yaml
action:
  - event: usee_notify
    event_data:
      kind: done
      title: Стирка
      message: Машинка закончила
```

## Исходник

`plugin.js` — чистый ES2020 без сборки (он же и дистрибутив). Собранная копия
в реестре обязана байт-в-байт совпадать по sha256 c `index.json`.
